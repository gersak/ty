Ty Icon Component - Attributes Demo

Size Attribute

Use the size attribute instead of classes:

<ty-icon size="xs">
<ty-icon size="sm">
<ty-icon size="md">
<ty-icon size="lg">
<ty-icon size="xl">
<ty-icon size="2xl">

Fixed pixel sizes:

<ty-icon size="16">
<ty-icon size="20">
<ty-icon size="24">
<ty-icon size="32">

Animation Attributes

Use spin and pulse attributes:

<ty-icon spin>
<ty-icon pulse>
<ty-icon spin class="text-important">

Combining Attributes with Classes

Attributes work together with CSS classes:

size="xl" class="text-positive"
size="2xl" spin class="text-negative"
size="lg" pulse class="icon-clickable"
