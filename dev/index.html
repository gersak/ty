Design System

Semantic Colors

Our design system uses semantic color naming:

Important Positive Negative Exception Neutral

CSS Variables

Everything is customizable through CSS variables:

:root { --ty-semantic-important: #3b82f6; --ty-semantic-positive: #10b981; --ty-semantic-negative: #ef4444; /* ... customize any semantic color */ }

Quick Start

1. Include the CSS Variables

<link rel="stylesheet" href="css/ty.variables.css">

2. Include the Library

<script src="/js/main.js"></script>

3. Use Components

<ty-button flavor="positive">Save</ty-button>
