Quick Start

1. Include the Library

<script src="/js/main.js"></script>

2. Use Components

<ty-button variant="positive">Save</ty-button>

3. Customize with CSS Variables

ty-button { --button-padding: 1rem 2rem; --button-font-size: 18px; }
